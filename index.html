Ola! Meu nome é João Fábio Pinto as pessoas costumam me chamar de Fábio, atuando no mercado de TI desde Janeiro de 2021, como testador de software de forma remota, onde adquiri vários conhecimentos como Analista de teste de software, com testes manuais e testes automatizados. Eu procuro sempre analisar um projeto, como telas, funcionalidades, critérios de aceite, regras de negócios que seriam implementados naquela aplicação para começar a fazer as escritas do meu plano de teste. Procuro sempre dar sugestões de que outros testes eu usaria para melhorar a aplicação no meu ponto de vista em qualidade, como usabilidade, qualidade entre outros.

Minhas Ferramentas de trabalho

Aqui estão as habilidades e ferramentas de Testes!

  • Desenvolvimento das automações

    Utilizo a IDE Eclipse para criar os projetos em Maven e Junit para os rodar os testes!

  • Cucumber

    Utilizo Cucumber, BDD e Gherkin para melhorar a colaboração, a clareza dos requisitos, a qualidade do código e a satisfação do cliente, resultando em um processo de desenvolvimento mais eficiente e eficaz.

  • Selenium

    Amplo conhecimento com automações utilizando o Selenium Web Driver.

  • Linguagem de Programação

    A linguagem de programação que eu uso é o Java, possuo conhecimentos também na Linguagem Python

  • APIs

    Utilizo o Postman para consumir e testar as APIs, fazer as validações dos EndPoints, Status code e Validações das mensagens de erro!

  • Mobile

    Conhecimentos com automações de mobile, Utilizando Android Studio e APPIUM para inspeção dos Elementos.

  • Jira Software

    Para Gerenciamento de Tarefas e Projetos utilizo o Jira Software.
